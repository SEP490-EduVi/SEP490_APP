EduVi Viewer
Hướng dẫn sử dụng dành cho Giáo Viên
Phiên bản 1.1.0  |  Windows
📋
EduVi Viewer là ứng dụng desktop giúp giáo viên mở và trình chiếu bài học (.eduvi) gồm hai loại: Bài trình chiếu và Trò chơi học tập. Ứng dụng chạy hoàn toàn offline — không cần Internet sau khi cài đặt.
1 Cài đặt ứng dụng
  • Trong thư mục này, tìm và chạy file EduViViewer-Setup-1.1.0.exe
    Nếu có file .zip, hãy giải nén trước rồi mới chạy file .exe.
  • Nếu Windows hiển thị hộp thoại cảnh báo "Windows protected your PC":
    → Bấm More info → Run anyway
    Đây là cảnh báo bình thường cho phần mềm chưa được ký chứng chỉ thương mại.
  • Trong bộ cài đặt: bấm Next → Install → Finish
    Ứng dụng sẽ được cài vào thư mục người dùng, không cần quyền Admin.
  • Sau khi cài xong, biểu tượng EduVi Viewer xuất hiện trên Desktop. Bấm đúp để mở.
✅
File .eduvi sẽ được liên kết tự động với EduVi Viewer sau khi cài đặt. Chỉ cần bấm đúp vào file .eduvi là mở ngay — không cần thao tác thêm.
2 Mở file bài học (.eduvi)

EduVi Viewer hỗ trợ 2 loại file đều có đuôi .eduvi:

Loại file Nội dung Cách mở
Slide Bài trình chiếu Các slide nội dung bài học với văn bản, hình ảnh, video, biểu đồ Mở trong ứng dụng, dùng phím → ← để chuyển slide
Game Trò chơi học tập Bài tập tương tác dạng trò chơi cho học sinh tham gia Ứng dụng tự nhận diện và mở giao diện game ngay lập tức
Ba cách mở file:
  • Kéo thả (nhanh nhất): Kéo file .eduvi từ File Explorer thẳng vào cửa sổ EduVi Viewer
  • Bấm đúp file: Bấm đúp vào file .eduvi bất kỳ (chỉ cần đã cài ứng dụng)
  • Nút Mở file: Trong ứng dụng, bấm nút Mở file hoặc nhấn Ctrl+O

4 Các loại trò chơi học tập

Hệ thống hỗ trợ 3 loại trò chơi. Ứng dụng tự nhận diện loại game từ file .eduvi và hiển thị giao diện phù hợp.

👆
Hover & Select
Học sinh di tay trước camera (MediaPipe) hoặc click chuột để chọn đáp án đúng trong 4 ô hiển thị trên màn hình.
Camera tay / Click chuột
🔀
Drag & Drop
Học sinh ghép nối các khái niệm với định nghĩa tương ứng bằng cách kéo thả hoặc click chọn rồi click vùng đích.
Kéo thả / Click ghép nối
🐍
Snake Duel
Trò chơi 2 người cùng màn hình. Mỗi người dùng bàn phím riêng để trả lời câu hỏi, thi điểm với nhau.
P1: phím 1 2 3 4  |  P2: phím Q W E R
🎮
Snake Duel cần 2 học sinh ngồi cùng một máy tính, mỗi người dùng một bên bàn phím.
Người chơi 1 (màu đỏ): phím 1 2 3 4
Người chơi 2 (màu xanh): phím Q W E R
5 Quy trình sử dụng điển hình trong lớp học
  • Trước giờ học: Nhận file .eduvi từ hệ thống EduVi (slide bài học hoặc game). Lưu vào máy tính của phòng học.
  • Bắt đầu tiết học: Bấm đúp vào file .eduvi để mở. Ứng dụng tự nhận diện đây là slide hay game.
  • Trình chiếu nội dung: Dùng phím → / ← chuyển slide. Nhấn F11 để bật toàn màn hình khi chiếu lên màn hình lớn.
  • Hoạt động game: Mở file game .eduvi tương ứng. Mời 1–2 học sinh lên tương tác trực tiếp với ứng dụng.
  • Kết thúc: Nhấn Esc để thoát về màn hình chính. Lịch sử file đã mở được lưu tự động.
6 Xử lý sự cố thường gặp
Vấn đềCách xử lý
Windows báo lỗi "Windows protected your PC" Bấm More info → Run anyway
Mở file .eduvi không có phản ứng Bấm phải file → Open with → chọn EduVi Viewer
Game mở ra nhưng không có nội dung File game có thể bị hỏng. Xuất lại file từ hệ thống EduVi và thử lại.
Hình ảnh / video trong slide không hiển thị Đảm bảo file .eduvi được xuất đầy đủ (có đính kèm asset). Không di chuyển hoặc đổi tên file sau khi xuất.
Camera không nhận diện tay (Hover Select) Kiểm tra camera có được kết nối và không bị ứng dụng khác chiếm dùng. Khởi động lại ứng dụng.
Ứng dụng bị treo hoặc không phản hồi Đóng ứng dụng bằng Alt+F4, mở lại bình thường. Nếu vẫn lỗi, liên hệ bộ phận kỹ thuật EduVi.
⚠️
Lưu ý quan trọng: File .eduvi là định dạng riêng của EduVi. Không thay đổi phần mở rộng file hoặc chỉnh sửa nội dung file bằng phần mềm khác — có thể làm hỏng dữ liệu.
7 Hỗ trợ

Nếu gặp vấn đề khi sử dụng, vui lòng liên hệ đội ngũ kỹ thuật EduVi để được hỗ trợ.
Cung cấp thông tin sau khi liên hệ:

  • Phiên bản ứng dụng: 1.1.0
  • Hệ điều hành Windows (Windows 10 / 11)
  • Mô tả ngắn gọn vấn đề gặp phải
  • Tên file .eduvi đang dùng (nếu liên quan đến file cụ thể)
