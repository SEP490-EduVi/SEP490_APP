# Flutter Player Workspace Module

This folder mirrors the target module boundaries for Flutter player decomposition:

- lib/core
- lib/features/game_library
- lib/features/game_launcher
- lib/features/offline_sync
- lib/data/local
- lib/data/remote
- lib/domain

Current production Flutter app remains at repository root (`lib/`) and already contains the offline core implementation added in this task. This folder is prepared for progressive migration.
